# Emet AI чат бот для умных новостных рекомендаций

### Emet это бот для мессенджера max, который подбирает новости под интересы пользователя, избегает информационных пузырей и помогает экономить время благодаря умной персонализации и краткими хайлайтами.

---
## Установка и запуск на ubuntu c docker compose
### Требуется установленный git и docker, расскасскажу про то что дополнительно надо
### Требуется в системе git lfs, установка под каждую систему своя, на Ubuntu
```bash
sudo apt-get update
sudo apt-get install git-lfs wget
git lfs install
```

Клонировать репозиторий:
```bash
GIT_LFS_SKIP_SMUDGE=1 git clone https://github.com/YatsenkoYura/max-emet-bot
cd max-emet-bot
curl -L -o models/fasttext_news_classifier.bin https://huggingface.co/data-silence/fasttext-rus-news-classifier/resolve/main/fasttext_news_classifier.bin
```
вообще модель есть на гите. Но у гита есть ограничение! так что надежнее скачать по wget как в гайде установки! но оно есть тут!
### Дальше нужно сделать копию .env.example и назвать ее .env
```
cp .env.example .env
nano .env
```
Вставьте в поля POSTGRES_PASSWORD свой пароль для бд. и на место TOKEN свой. Сохраните и выйдете.
Дальше все просто
```
docker compose up --build
```

## Основная идея

Emet анализирует реакции пользователя (лайки, дизлайки, скипы), формирует профиль интересов и предлагает новости, соответствующие предпочтениям. Алгоритм строится на системе весов категорий, пользовательских взаимодействиях и оценке релевантности.

---

## Ключевые возможности

* **Персональные рекомендации** на основе поведения и предпочтений.
* **Подборка похожих новостей** - подбор новости по промпту из разных источников.
* **Отслеживание реакций пользователя:** .
* **Подбор новотей по категориям**.
* **поиск новостей по ключевым словам**.

---

## Архитектура проекта

Проект включает несколько ключевых модулей:

### 1. **Модуль рекомендаций**

* Персонализация на основе весов категорий.
* Динамическое пересчитывание интересов.
* Учёт уверенности по категориям.
* Ранжирование новостей по score.

### 2. **Модуль MAX‑бота**

* Обработчики сообщений.
* Выдача новостей.
* Поиск новостей
* Обработка реакций.

### 3. **Модуль интеграции с источниками новостей**

* Парсеры или API.
* Очистка, нормализация, определение категории.
---

## Целевая аудитория

Продукт рассчитан на широкую аудиторию, но включает 4 основных сегмента:

* **Профессионалы** - ценят скорость и точность.
* **Молодые люди** - хотят краткости.
* **Энтузиасты тем** - выбирают любимые категории.
* **Осознанные читатели** - избегают инфопузырей.
